## Day2：Selenium 自动化测试

### 第一步：启动登录页面

```bash
# 终端 1
python test/day2/login_page.py
```

启动后访问 http://127.0.0.1:7861

### 第二步：运行 Selenium 测试

```bash
# 终端 2
cd test/day2/selenium
pytest -v
```

### 输出结果

| 文件 | 说明 |
|------|------|
| `test/day2/selenium/test_results.json` | 测试结果（JSON，30 条） |

## 常用命令速查

```bash
# Day1 生成用例
python test/day1/app.py

# Day2 启动登录页面
python test/day2/login_page.py

# Day2 运行测试
cd test/day2/selenium && pytest -v

# 查看测试结果
cat test/day2/selenium/test_results.json

# 查看生成的用例
cat test/day1/output/testcases.md

# 停止登录页面
pkill -f login_page.py
```
