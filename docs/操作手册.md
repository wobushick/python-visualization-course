## Day2：Selenium 自动化测试

### 运行测试

```bash
cd test/day2/selenium
pytest -v
```

测试会自动启动 HTTP 服务器提供登录页面，无需手动启动。

默认以**有头模式**运行 Chrome，可直观看到浏览器操作过程。如需无头模式（CI/服务器环境）：

```bash
HEADLESS=1 pytest -v
```

### 查看登录页面

直接浏览器打开 `test/day2/login.html` 即可预览。

### 输出结果

| 文件 | 说明 |
|------|------|
| `test/day2/selenium/test_results.json` | 测试结果（JSON，30 条） |

### 示例输出


## 常用命令速查

```bash
# Day1 生成用例
python test/day1/app.py

# Day2 运行测试（有头模式，默认）
cd test/day2/selenium && pytest -v

# Day2 运行测试（无头模式）
cd test/day2/selenium && HEADLESS=1 pytest -v

# 查看测试结果
cat test/day2/selenium/test_results.json

# 查看生成的用例
cat test/day1/output/testcases.md
```
